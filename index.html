1. GOODとBADを登録

それぞれ文字を書きながら10秒間撮影します。 写真・動画ではなく、正規化した手の座標だけを端末内に保存します。

GOOD 正しい持ち方で書く
BAD 一番よく戻る持ち方で書く

BAD登録では、無理に複数の崩れ方を混ぜず、まず一番よく起きる持ち方を登録してください。

2. 見守り

判定方法と個人データについて

現在の手の形を、登録したGOOD群とBAD群の両方に比較し、どちらへ近いかを判定します。 BAD寄りの状態が約3秒続いた場合だけ通知します。

カメラ映像はブラウザ内で処理し、サーバーへ送信しません。 保存するのは手の形を数値化した座標データだけです。 このページには解析・広告・外部ログ送信のコードを入れていません。

GitHub Pagesを開く通信と、アプリ本体・AIモデルの読み込み通信は発生します。 初回読み込み後は、対応ブラウザでは必要ファイルをキャッシュします。
